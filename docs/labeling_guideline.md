# CVS Radar 人工標註規範

本規範用於建立可離線重現的 gold labels，讓後續能客觀比較「現有規則」與「LLM」在跨品牌/比較留言判斷與留言情感上的表現。標註單位是一則留言，不是整篇貼文。

## 欄位定義

每列 CSV 代表一則留言，標註者只填下列欄位：

| 欄位 | 可填值 | 說明 |
|---|---|---|
| `sentiment` | `正` / `負` / `中性` | 留言對主要討論對象的情感極性。 |
| `target_brand` | `本牌` / `他牌:<品牌>` / `無` | 這句主要在講哪個品牌。`本牌` 指貼文品牌；他牌請寫實際品牌，例如 `他牌:7-11`。 |
| `is_comparative` | `是` / `否` | 是否有明確比較兩個以上品牌、通路、商品或選項。 |
| `favored_brand` | `本牌` / `他牌` / `平手` / `不明` | 只有比較句才判斷誰勝。非比較句填 `不明`。 |
| `notes` | 自由文字 | 可記錄反諷、脈絡不足、多品牌混雜等疑點。 |

## sentiment 判準

`sentiment` 標的是留言對「主要討論對象」的態度，不一定等於貼文品牌。

- `正`：明確稱讚、推薦、回購、覺得好吃/划算/方便，或比較後認為某品牌較好。
- `負`：明確抱怨、反推、踩雷、太貴、難吃、份量差，或比較後認為某品牌較差。
- `中性`：只陳述資訊、詢問、無足夠情緒、正負混雜但沒有明確結論。

### sentiment 例子

| 留言 | sentiment | 理由 |
|---|---|---|
| `好吃，會回購` | `正` | 明確正向。 |
| `太乾，踩雷` | `負` | 明確負向。 |
| `哪家還買得到？` | `中性` | 詢問庫存，沒有評價。 |
| `普通但方便` | `中性` | 正負混合且強度低。 |
| `雖然貴，但比小7好吃` | `正` | 對本牌的結論是勝出。 |
| `哇，這價格真是佛心到我不敢買` | `負` | 反諷，實際態度是價格太高。 |

反諷以「實際語意」標，不以表面正向詞標。若無法判斷是否反諷，標 `中性` 並在 `notes` 說明。

## target_brand 判準

`target_brand` 標這句主要評論的品牌：

- `本牌`：留言主要在講貼文品牌或貼文商品，即使沒有明說品牌。
- `他牌:<品牌>`：留言主要在講非貼文品牌，例如貼文是全家，留言說 `小7比較便宜`，標 `他牌:7-11`。
- `無`：純詢問、玩笑、無品牌指向、只講個人狀態，或空留言。

若一句同時提到多品牌，選「留言評價重心」或「結論指向」的品牌。真的無法判斷時填 `無`，並在 `notes` 註明。

## is_comparative 判準

`is_comparative=是` 需要有明確比較語意，例如：

- 比、比較、較、勝過、輸、贏、不如、屌打、還是 X 好、沒有 X 好
- 以價格、份量、味道、CP 值等面向比較兩個品牌或商品
- 不一定要同時寫出兩個品牌；若脈絡明確用貼文品牌對他牌比較，也算比較句

不是比較句的例子：

- `小7也有賣`：提到他牌，但沒有比較誰好。
- `全家這款好吃`：只評本牌。
- `OK 上週特價`：資訊陳述，沒有比較。

## favored_brand 判準

只有 `is_comparative=是` 時才判斷：

- `本牌`：比較後貼文品牌勝出，例如貼文是全家，留言 `雖貴但比小7好`。
- `他牌`：比較後非貼文品牌勝出，例如貼文是全家，留言 `小7比全家便宜好吃`。
- `平手`：明確表示差不多、各有優缺、沒有勝負，例如 `跟小7差不多`。
- `不明`：有比較語氣，但看不出誰勝，例如 `有人比較過全家跟小7嗎？`。

非比較句一律填 `不明`。

## 正例、反例與邊界案例

| 貼文品牌 | 留言 | sentiment | target_brand | is_comparative | favored_brand | 說明 |
|---|---|---|---|---|---|---|
| 全家 | `雖然貴，但比小7好吃` | `正` | `本牌` | `是` | `本牌` | 轉折後結論支持本牌。 |
| 全家 | `小7比全家便宜好吃` | `正` | `他牌:7-11` | `是` | `他牌` | 他牌勝出。 |
| 全家 | `小7也有賣` | `中性` | `他牌:7-11` | `否` | `不明` | 提到他牌但沒有比較。 |
| 7-11 | `好棒喔，貴到可以當精品` | `負` | `本牌` | `否` | `不明` | 反諷，標實際負向。 |
| 全家 | `全家跟小7差不多，各有優缺` | `中性` | `無` | `是` | `平手` | 明確平手。 |
| 全家 | `全家好吃，小7也不錯，OK 沒吃過` | `正` | `本牌` | `否` | `不明` | 多品牌混雜但沒有明確比較勝負。 |
| 全家 | `比 OK 便宜，但味道輸小7` | `負` | `他牌:7-11` | `是` | `他牌` | 多品牌比較，以最後味道結論為主；若任務要拆面向，另行加欄。 |
| 萊爾富 | `有人吃過跟小7比如何？` | `中性` | `無` | `是` | `不明` | 問句，有比較意圖但無勝負。 |

## 標註原則

1. 先看 `context`、`post_brand`、`product_name`，再看留言。
2. 以留言本身為主；不要用標註者自己的品牌偏好補腦。
3. 品牌別名請歸一：`711`、`小7`、`小七` 都視為 `7-11`；`FamilyMart`、`fami` 視為 `全家`；`Hi-Life` 視為 `萊爾富`。
4. 無法判斷時選保守值：情感選 `中性`，品牌選 `無`，勝負選 `不明`，並寫 `notes`。
5. 若一句話包含多個可分離主張，但 CSV 只有一列，標「主要結論」。後續若要更細，可升級成 span-level 或 aspect-level 標註。
